Movie Review database

POPULATED DATABASE

RULES:
1)For select commands do not use *, write the column names.
2)Attributes like movieID, reviewId,title,director,year and review date are of type varchar and stars are of type int.
3) Each question carries 2 marks, this set is for 10 marks.

DATABASE SCHEMA

ANSWERS

QUESTION 1
Find the titles of all movies directed by Giancarlo.


SELECT movieID,title,year,director FROM movieT WHERE director= “GIANCARLO”;


QUESTION 2
Find all years that have a movie that received a rating of 4 or 5, and sort them in increasing order.


SELECT ratingDate FROM rating WHERE stars>3 ORDER BY stars;


QUESTION 3
Find the titles of all movies that have rating = 2.


SELECT title FROM movieT M,rating R WHERE R.movieId=M.movieID AND R.stars=2;


QUESTION 4
Show reviewers who gave ratings in the year of 2019.


SELECT reviewID FROM rating WHERE ratingDate like “%2019%”;


QUESTION 5
Return all reviewer id’s and movie names together in a single list, alphabetized.


SELECT reviewId,title FROM movieT M, rating R WHERE M.movieID = R.movieId ORDER BY title ASC;
